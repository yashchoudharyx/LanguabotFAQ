What is LinguaPal?

LinguaPal is a language-learning chatbot that helps users translate basic phrases and identify the language of given greetings.

What languages does LinguaPal support?

English, French, Spanish, Japanese, German, and Hindi.

Can I use LinguaPal to learn full conversations?

No, it is designed for short, common phrases and greetings.

How accurate is LinguaPal's language identification?

It uses static keyword-based matching, so it can recognize common greetings in supported languages.

Does LinguaPal use any external APIs like Google Translate?

No, LinguaPal uses a fixed set of predefined phrases and does not rely on external translation APIs.

How do I ask the bot to translate a phrase?

You can say things like How do I say thank you in French? or Translate good night to Spanish.

What phrases can LinguaPal translate?

Hello, Thank you, Good morning, Good night, and Goodbye.

Can I ask for multiple translations at once?

No, LinguaPal handles one phrase-language pair per request.

How does LinguaPal know what I want to translate?

It extracts the phrase and target language from your message using Dialogflow slot filling.

What happens if I ask to translate an unsupported phrase?

The bot responds with I’m still learning that one!

Can LinguaPal tell me what language I’m speaking?

Yes. For example, if you type Hola, it will respond That’s Spanish! It means hello.

How many languages can it identify?

LinguaPal can identify greetings in six supported languages.

What happens if I type something in a language it doesn’t support?

The bot will respond with I’m not sure which language that is yet.

Can LinguaPal identify entire sentences?

No, it is designed to detect short greetings and keywords.

Can LinguaPal teach me common greetings?

Yes. Try asking Teach me some greetings in Japanese or What’s hello in German?

What types of phrases can I learn?

Greetings, polite expressions like thank you, and common goodbyes.

Will it teach me how to pronounce the words?

It includes simple pronunciation text, but does not support audio.

Can it give cultural tips?

No, the current version focuses only on basic phrase learning.

Is this a real translation tool?

No, it is a demo project built to showcase Dialogflow CX conversational design.

Can it handle spelling mistakes?

Minor errors may be understood by Dialogflow’s natural language engine.

Can I use voice input with LinguaPal?

Yes, if deployed on a platform that supports voice like Google Assistant.

How can I extend LinguaPal?

You can add real translation APIs or expand to more languages and phrases.

Does it store any user data?

No, LinguaPal is stateless and does not store user history.

What should I do if the bot doesn’t respond correctly?

Try rephrasing your input or use simpler wording like Translate hello to Spanish.

Why does the bot say I’m still learning?

This is a fallback response when a phrase or language is not supported.
